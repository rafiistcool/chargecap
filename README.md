# ChargeCap ⚡

Lightweight macOS menu-bar app that limits your MacBook's battery charge to extend its lifespan.

## Why?
Keeping your MacBook at 100% charge degrades the battery. ChargeCap lets you set a maximum charge limit (e.g. 80%) and stops charging automatically.

## Features
- 🎯 Custom charge limit (20-100%)
- 📊 Live battery status in menu bar
- 🚀 Launch at login
- 🪶 Lightweight, native SwiftUI

## Pricing
- **Free:** Battery info + charge alerts
- **Pro ($4.99):** Automatic charge limiting + scheduling
