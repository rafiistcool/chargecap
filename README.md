# ChargeCap ⚡

Lightweight macOS menu-bar app that limits your MacBook's battery charge to extend its lifespan.

[![License: BSL-1.1](https://img.shields.io/badge/License-BSL--1.1-blue.svg)](LICENSE)

## Why?
Keeping your MacBook at 100% charge degrades the battery. ChargeCap lets you set a maximum charge limit (e.g. 80%) and stops charging automatically.

## Features

### Free & Open Source
- 🔋 Live battery monitoring (charge %, health, cycles, temperature)
- 📊 Battery health status with color indicators
- 🔔 Charge alerts and notifications
- 🎯 Menu bar integration with dynamic status

### Pro ($4.99 lifetime)
- ⚡ Charge limiting (stop at any percentage)
- ⛵ Sailing mode (auto discharge/recharge within range)
- 🌡️ Heat protection (pause charging at high temps)
- 📅 Charge scheduling (top up before a trip)

## Install
Download the latest release from [Releases](https://github.com/rafiistcool/chargecap/releases).

Requires macOS 13+ (Ventura or later). Works on Intel and Apple Silicon.

## Build from Source
```bash
git clone https://github.com/rafiistcool/chargecap.git
cd chargecap
open ChargeCap.xcodeproj
# Build & run in Xcode (⌘R)
```
